# 仿生贝壳粉底盒作品集设计文档

## 项目定位

这是作品集第二个项目，建议替换当前占位的 `Field Lamp`。项目以海扇贝 / 扇贝壳为外形母题，把便携粉底盒、补妆镜、粉扑收纳和装饰属性整合成一个偏趣味礼物感的化妆品容器。

- **推荐中文标题：** 仿生贝壳粉底盒
- **推荐英文标题：** Haitober Shell Compact
- **项目类型：** 趣味装饰便携粉底盒 / 化妆品容器概念
- **一句话概括：** 以贝壳开合形态组织镜子、粉盘与粉扑，让补妆动作带有“打开贝壳”的仪式感。
- **角色：** 产品设计 / 外观设计 / CMF 表达
- **工具：** Sketching, Rhino / Blender, KeyShot, Rendering
- **适用场景：** 日常通勤、假期出游、桌面装饰、礼品化妆品包装
- **关键词：** Biomimicry, Cosmetic Packaging, Portable Object, CMF, Decorative Function

## 核心叙事

这个项目最值得强调的不是“做了一个像贝壳的粉底盒”，而是“把贝壳的开合、层叠、珍珠意象转化成具体的化妆品使用结构”。上盖承担镜子，贝壳内腔承担粉盘和粉扑，中心圆球既是视觉焦点，也暗示粉扑把手 / 珍珠装饰。整体适合放在作品集里作为鼠标项目之后的第二个案例：它比鼠标更偏 CMF 和情绪价值，可以补足作品集里的“消费品包装与装饰性产品”能力。

## 设计问题

普通粉底盒常常是扁平圆盒，功能清楚但记忆点弱；而过度装饰的化妆品容器又容易牺牲便携和收纳逻辑。这个项目需要在三件事之间取得平衡：

- 外形必须有明确的贝壳识别度，而不是普通花瓣或玩具形态。
- 镜子、粉盘、粉扑需要在一个小体积里有清楚的层级。
- 产品要有可爱和礼品感，但不能让结构显得脆弱、繁琐或难清洁。

## 设计目标

- **形态识别：** 用扇贝的放射肋、波浪外缘和开合姿态建立第一眼记忆点。
- **便携结构：** 将粉盘、镜子、粉扑和限位件整合在上下壳体之间，减少额外配件。
- **补妆体验：** 打开动作像打开贝壳，镜面角度面向使用者，中心粉扑便于拿取。
- **装饰属性：** 闭合时像一个小贝壳摆件，打开时像展示珍珠，适合桌面和随身拍摄。
- **CMF 表达：** 白瓷感外壳、浅粉内盘、局部金属件和柔软粉扑形成“海洋 + 美妆”的轻奢趣味。

## 形态与结构拆解

### 上盖

上盖外侧采用贝壳放射肋纹，外轮廓保留圆润波浪边。内侧嵌入小型化妆镜，镜框顺着贝壳边缘形成柔和包边。作品集里可以强调：上盖不是简单贴一个镜子，而是利用贝壳开合的自然姿态，把镜子角度变成使用场景的一部分。

### 下盖

下盖是主要容器。浅粉色贝壳内盘承载粉盒，外围白色壳体提供保护和视觉边界。底部可作为粉盒主体，闭合时保护粉盘，打开时成为稳定底座。

### 中心粉扑 / 珍珠件

中心白色圆球是视觉焦点，建议在作品集叙事中解释为“珍珠化的粉扑把手”。它既强化海洋意象，也让使用者有明确的抓取点。若后续细化实物逻辑，可以将圆球与粉扑底座做成可拆卸结构。

### 限位与转轴

图中有金属转杆、陶瓷限位器等结构暗示。作品集里应把这部分写成“初步结构设想”，避免过度承诺量产可行性：金属转杆控制开合，陶瓷 / 塑料限位件支撑镜面角度，内部粉盘可拆卸，便于清洁和替换。

3. **三视图与结构：** 展示正视、侧视、俯视，说明上盖、下盖、镜子、粉盘、粉扑、转轴的关系。

6. **情绪化展示：** 用粉色台阶、柔光、桌面场景呈现礼品感和装饰性。

## 页面文案草稿

### 短介绍

这款便携粉底盒以贝壳为仿生原型，将开合动作、镜面角度、粉盘收纳和珍珠意象整合在一个小型化妆品容器中。白瓷感外壳提供干净的礼品感，浅粉内盘和圆形粉扑让补妆动作更轻松，也让产品在桌面上具有装饰属性。

### 过程说明

设计从“如何让粉底盒更有记忆点”开始，而不是直接复制贝壳外形。贝壳的波浪边缘被转化为上下盖轮廓，放射肋纹被转化为外壳表面起伏，打开贝壳的姿态则被用来组织镜子、粉盘和粉扑的位置关系。

### 结构说明

上盖内嵌小镜子，下盖承载粉盘和可拆卸粉扑。中心圆球借用珍珠意象，既是视觉焦点，也可以作为粉扑抓取提示。金属转杆和限位件控制开合角度，使镜子在补妆时保持可观察的位置。

### 反思

这个项目让我意识到，仿生设计不应该停在外形相似。更重要的是找到自然形态背后的动作逻辑：贝壳的开合、珍珠的中心意象、放射纹理的方向感，都可以被转译成产品结构和用户体验。

## 与当前作品集风格的衔接

当前网站是深色 graphite 工业设计实验室风格。贝壳粉底盒素材偏粉色和美妆海报感，接入时建议不要让整个页面变成粉色主题，而是在深色基底里使用粉色图片作为项目自身的 CMF 信号。

- 背景继续使用 graphite / charcoal。
- 标注色可以保留 titanium green，但给贝壳项目新增少量 shell pink。
- 图片外框不做大圆角，保持当前作品集的克制实验室感。
- 文案重点从“可爱”转向“仿生转译、结构收纳、情绪价值”，这样更适合面试讲述。
