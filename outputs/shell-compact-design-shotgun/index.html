Design Shotgun / 02 Shell Compact Portfolio Direction

仿生贝壳粉底盒

把“打开贝壳”变成补妆动作。

这不是一个粉色装饰页，而是第二个作品集案例的视觉打法选择： 它要补足鼠标项目之外的 CMF、消费品包装、情绪价值与小型结构叙事。

CEO Review 关键修正

从“造型像贝壳”升级为“镜面角度、粉扑拿取、粉盘保护、卫生收纳和装饰身份被同一个开合结构组织”。

Three Directions

三套落页设计方向

A / Lab Case

设计实验室式短案例

最贴合当前作品集。把粉色素材放进 graphite 实验室系统里，用标注、结构拆解和小图矩阵压住甜美感。

Graphite Titanium Green Shell Pink as signal
首屏

左侧一句话洞察，右侧打开状态大图；下方露出结构分解卡片。

页面节奏

Hero → Insight → Part stack → Details → Reflection，控制在短案例长度。

最强点

和鼠标项目风格统一，适合直接并入当前 Next.js 作品集。

B / Beauty Editorial

美妆编辑式情绪页面

把项目当作小型美妆品牌海报处理。大面积留白、粉色台阶、柔光静物，让作品更有商业包装感。

Blush Pink Warm Porcelain Editorial Poster
首屏

竖版海报图做主视觉，标题像香水/美妆 campaign，弱化技术感。

页面节奏

Hero → Lifestyle → CMF → Details → Image2 补图建议。

风险

和当前作品集深色工业风割裂，需要外层框架拉回专业感。

C / Anatomy Board

产品结构图谱式案例

用爆炸图、三视图、零件标签和补妆动作链条来讲。它最能修复 CEO review 里“结构可信度不足”的问题。

Part Stack Use Ritual Annotation-heavy
首屏

大标题下直接给“开合结构如何服务补妆”的四步流程。

页面节奏

Problem → Use flow → Exploded view → Component details → Missing asset plan。

风险

现有真实过程素材不足，最好补一张结构剖面或用 image2 生成辅助图。

Recommendation

推荐采用 A + C 的混合方向

用 A 的深色实验室外壳保持作品集一致性，用 C 的结构图谱补强说服力。 B 的美妆情绪只保留为局部图像，不让它接管整个页面。

首屏

标题：仿生贝壳粉底盒。副标题：以贝壳开合结构组织镜面、粉盘和粉扑，让补妆动作变成可展示的小型仪式。

核心模块

洞察卡片、四步使用流程、零件层级、CMF 材料条、现有图像图库、AI 补图提示。

下一步实现
